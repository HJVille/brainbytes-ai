# BrainBytes AI Tutoring Platform

## Project Overview
BrainBytes is an AI-powered tutoring platform designed to provide accessible academic assistance to Filipino students. This project implements the platform using modern DevOps practices, containerization, CI/CD automation, cloud deployment, and monitoring.

## Project Goals
- Implement a containerized application with proper networking
- Create a frontend container using Next.js
- Create a backend container using Node.js
- Connect the application to MongoDB Atlas
- Create an automated CI/CD pipeline using GitHub Actions
- Deploy the application to Oracle Cloud Free Tier
- Set up monitoring and observability tools using Prometheus and Grafana
- Document the setup process for team collaboration

## Technology Stack
- Frontend: Next.js
- Backend: Node.js
- Database: MongoDB Atlas
- Containerization: Docker
- Multi-container Setup: Docker Compose
- CI/CD: GitHub Actions
- Cloud Provider: Oracle Cloud Free Tier
- Monitoring: Prometheus and Grafana
